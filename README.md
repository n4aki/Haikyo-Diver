# 廃墟ダイバー

崩壊した研究施設を探索し、酸素が尽きる前に出口を見つけて下層へ進んでいく、ターン制 2D グリッド型ローグライト寄りローグライクです。  
HTML / CSS / JavaScript だけで動作し、ビルド不要で GitHub Pages にそのまま配置できます。

## ゲーム概要

- 舞台はダーク SF 風の廃研究施設
- 1 ターンごとにプレイヤーと敵が行動するターン制
- 酸素は 2 行動ごとに 1 減少し、0 になると HP が減少
- フロアを探索して階段を見つけ、強化を選びながら先へ進む
- 5F / 10F はボスフロア
- 10F のボス撃破で仮クリア

## ローカルでの起動方法

ビルドは不要です。以下のどちらかで起動できます。

1. `index.html` を直接ブラウザで開く
2. 簡易サーバーを立てて開く

```bash
python -m http.server 8000
```

その後、ブラウザで `http://localhost:8000` を開いてください。

## GitHub Pages で公開する手順

1. このディレクトリを GitHub リポジトリへ push する
2. GitHub の `Settings` → `Pages` を開く
3. `Source` を `Deploy from a branch` にする
4. Branch は `main`、Folder は `/ (root)` を選ぶ
5. 保存後、数分待つと公開 URL が発行される

`index.html` がルートにある構成なので、そのまま GitHub Pages で動作します。

## 現在の仕様

### フロア進行

- 通常フロアはランダム生成
- ボスフロアは 5F / 10F
- 10F のボスを倒すとクリア
- 階段到達時に 3 択アップグレード
- ボス撃破時に攻撃特性選択 + 全回復

### マップ

- マップ全体は `26x26`
- 画面表示は `10x10` ビューポート
- カメラはプレイヤーに追従してスクロール
- 通常フロアは 4〜8 部屋程度の矩形部屋 + 通路
- 部屋は `通常 / 物資 / 危険` の内部タイプを持つ
- 開始地点から階段まで到達可能になるよう接続
- 階段は通路に隣接しない位置を優先して配置

### 視界と探索

- 未探索 / 探索済み / 現在視界内 を分けて表示
- 通路では視界が狭い
- 部屋では部屋全体が見える
- 敵やアイテムは現在視界内のみ表示

### プレイヤー

- 初期値
  - HP: 50
  - 酸素: 30
  - 攻撃力: 3
- 移動は上下左右 1 マス
- `待機` 可能
- プレイヤー表示は PNG スプライトアニメーション
  - `idle`
  - `walk`
  - `attack`
- 攻撃時は前方へ踏み込む演出あり
- 被弾時はのけぞり演出あり

### 攻撃特性

プレイヤーは常に 1 つだけ攻撃特性を持ちます。  
初期は `標準攻撃` で、ボス撃破時に新しい攻撃特性へ切り替えます。

- 標準攻撃: 隣接 1 マスを攻撃
- 貫通ショット: 直線上の複数敵を攻撃
- 扇状ショット: 近距離の複数マスを攻撃
- 爆発弾: 着弾点周囲へ範囲攻撃
- 単体高火力ショット: 単体へ高ダメージ

### アップグレード

フロア突破時にラン中アップグレードを獲得できます。主なものは以下です。

- 攻撃力アップ
- 最大 HP アップ
- 最大酸素アップ
- 撃破時 HP 回復
- 撃破時 酸素回復
- クリティカル率アップ
- 追撃弾
- ルームシールド
- 省エネ呼吸

### 敵

- スライム: 基本の近接敵
- 警備ロボット: 遅いが硬い重装敵
- ハウンド: 2 マス詰める高速敵
- アイボット: 直線射撃を行う遠距離敵
- ボマー: 接近時 / 撃破時に爆発
- メディック: 周囲の敵を回復
- ベヒモス: ボスフロアに出現する高耐久ボス

フロア帯によって出現傾向が変わります。

- 1〜2F: スライム / ハウンド / 警備ロボット中心
- 3〜4F: アイボットが追加
- 5F 以降の通常フロア: ボマー / メディックも追加

### アイテム

- 応急キット
  - 最大 HP の 10% 回復
- 酸素ボンベ
  - 最大酸素の 20% 回復
- 敵撃破時に確率でアイテムドロップあり
- 初期配置アイテム数は控えめ

### 戦闘演出

- 攻撃先マスの短い発光エフェクト
- プレイヤー攻撃はシアン系、敵攻撃は危険色寄り
- ダメージ / 回復時に頭上へ floating text
- 酸素切れ時にも一時メッセージ表示

### UI / 操作

- PC
  - 矢印キー
  - `WASD`
  - `Space` で待機
- スマホ
  - 画面下の仮想方向ボタン
  - 待機ボタン
- ログはマップ右下の小さいオーバーレイ
- フロア開始時に大きい階層表示を短時間表示

## アセット構成

### プレイヤーアニメーション

プレイヤーは `assets/sprites/player/` 配下の PNG を使用します。

- `idle_down.png`
- `idle_left.png`
- `idle_right.png`
- `idle_up.png`
- `walk_down.png`
- `walk_left.png`
- `walk_right.png`
- `walk_up.png`
- `attack_down.png`
- `attack_left.png`
- `attack_right.png`
- `attack_up.png`

各ファイルは `1 アクション × 1 方向` で、基本は 4 フレームの `2x2` グリッドです。

  - `assets/items/medkit.png`
